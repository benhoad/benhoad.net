# How to: Stop CSS animation flicker in Webkit

 - published: 2012-05-19 20:10:00
 - tags: css

If you're trying out some CSS3 animations but are noticing weird font flickering in Safari or Chrome then you've come to the right place. All you need to do is add this CSS rule to any element that you are seeing the flicker on:

    -webkit-transform-style: preserve-3d;

And that's it.
