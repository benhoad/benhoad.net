# NPM: React Loading Animation

- published: 2016-05-01 23:00
- tags: javascript

I just published a new NPM for [showing a loading spinner](https://www.npmjs.com/package/react-loading-animation) 
while your React content is loading.

To use it just wrap your load-dependent content with it:

<script src="https://gist.github.com/nathanhoad/7ba40d73bae9ca6795774306f8a6a92f.js"></script>
